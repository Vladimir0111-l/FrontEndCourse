На главную HTML CSS JavaScript
HTML это
  • Создание своих и редактирование чужих страниц и сайтов.
  • Настройка готовых сайтов и шаблонов.
  • Контроль внешнего вида своих материалов при их публикации на разных сайтах.
  • Возможность использовать чужой макет или фрагменты кода в своих проектах.
  • Обход некоторых барьеров и блокировок доступа к закрытому контенту.
CSS позволит
  • Оформление текста: задание цвета, размера шрифта, выравнивания, высоты строки и т. д.
  • Оформление html-элементов: размеры, отступы, цвет фона, тени, скругления.
  • Построение сеток для расположения контента.
  • Создание анимации без использования JavaScript.
  • Создание адаптивного дизайна: возможность менять отображение в зависимости от размеров устройства, на котором просматривают сайт.
JavaScript поможет в следующих задачах
  • Создание веб-приложений: почтовых клиентов, редакторов текста, социальных сетей, видеохостингов и других.
  • Разработка бэкенда веб-приложений: взаимодействие с базами данных.
  • Создание браузерных расширений: блокировка рекламы, скачивание видео, подсчёт времени, проведённого в интернете.
  • Разработка игр: браузерные бродилки, инди-проекты.
  • Машинное обучение: создание веб-приложений, визуализирующих данные из моделей машинного обучения.
