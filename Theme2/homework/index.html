Введение в фронтенд разрабтоку

Фронтенд разработка отвечает за создание интерфейса пользовательской части веб сайта или веб-приложения. Она включает в себя работу с HTML, CSS, JavaScript.


Языки фронтенд разработки

Oсновнымим языками Фронтенд разработки являются:

  • HTML - используется для структурирования веб-страницы;
  • CSS - используется для стилизации веб веб-страницы и оформления ее внешнего вида;
  • JavaScript - используется для добавления интерактивности на веб-страницу, такой как анимации, обработка событий и динамическое обновление содержимого.

Инструменты фронтенд разработки

  1. VS Code - Мощный редактор кода с поддержкой множества языков и плагинов;
  2. React - библиотека для создания пользовательсих интерфейсов;
  3. Sass - репроцессор CSS с расширинными возможностями;
  4. Webpack - Сборщик модулей для JavaScript приложений.
